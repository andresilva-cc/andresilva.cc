Running catalog of composition levers across the 19-slot redesign set for andresilva.cc. Use before drafting new directions.

**Why:** The rejected list in MEMORY.md tells you what's forbidden, but not what's occupied. Without a ledger, new directions quietly duplicate levers of kept neighbors — e.g. another "grid of identical cards" when 01/07/17 already own it.

**How to apply:** Before drafting, cross-reference each slot's primary lever here. If a new idea shares a lever with any kept direction, pick a different lever or sharpen the difference until it's compositionally distinct.

## Rejected composition patterns (do not reattempt)

- Timeline as primary axis (date-rail)
- Any split-screen of any ratio (identity-vs-content)
- Modern serif display on dark
- Whitespace as architecture (cathedral)
- Full-bleed color blocking
- Data-viz graph as home hero
- Interaction-gated disclosure (reveal on focus/hover)
- Object-impersonation metaphors (library, terminal, IDE, newspaper, etc.)
- Editorial-magazine cosplay (drop caps, chapter numerals)

## Open lever territory (unoccupied, rule-file compliant)

- Long-form scroll narrative (Home as a single downward document, chaptered not time-ordered)
- Small-multiples (uniform identical tiles/cells forming rhythm — distinct from Info Block's 3 sizes because every tile is the same size)
- Toolbar + results (persistent sticky filter/sort/view bar driving a results region — distinct from Control Panel's heterogeneous dashboard because it's a single uniform result list)
- Annotated cards (each card is a 100+ word mini-essay, not just title+chips)
- Oversized metadata (dates/years/tech names as primary display typography, inverting the normal visual hierarchy)
- Strict scales with unusual step counts (5-size, golden-ratio, musical scale)
- Inline-linked prose bio (bio paragraph where every noun is a link — different from Marginalia's gutter because links are in the sentence)
